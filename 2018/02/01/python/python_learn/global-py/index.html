什麼是命名空間

每一個 function or class 等等都有定義自己的命名空間。
假如你在 fun1( ) 有一個 variable 叫做 X，又在 fun2( ) 裡面有個 variable 也叫做 X，
雖然名稱一樣，但是他們對應的卻是不一樣的東西，而且也不跨界使用。
