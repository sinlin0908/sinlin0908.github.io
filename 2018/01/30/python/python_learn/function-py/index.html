參數值使用 *

在 C C++ 裡 * 表示 pointer ，但 Python 卻不是喔!
那是什麼?
如果你在參數中加入 * ，他會把你傳到 function 的資料集合起來，變成一個 Tuple 型態。

也就是一個參數可以包含很多 data。

在 Python 的世界裡面所有東西都是一個 Object。
因此，你可以把 function 傳給一個 variable，把他們當作參數傳給另一個 function。
